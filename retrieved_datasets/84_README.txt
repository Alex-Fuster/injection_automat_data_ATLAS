Data and file overview
1. A short description of what data it contains:
The file 'database_Anticosti_vegetation_clearcuts.xlsx' contains all the original vegetation data sampled from 1995 to 2011 in large clearcuts on Anticosti Island (see Article: Beguin, J., Côté, S.D., Vellend, M. 2022. Large herbivores trigger spatiotemporal changes in forest plant diversity. Ecology)  
The file contains two separate sheets:
 - The first sheet labeled 'plot-species' contains: 1) the unique ID of each sample plot (first column) and 2) the percentage of horizontal cover for each taxon (all other columns) 
 - The second sheet labeled 'plot-treatment' contains: 1) the unique ID of each sample plot (first column) and 2) the details about the different experimental units, treatment (Exclosure) and year 
2. The two sheets can be joined to one another using the first colomn labeled 'IdPlot'.
3. Date that the file was created: 5 January 2021.

* Second sheet labeled 'plot-treatment':
1. Count of number of variables, and number of cases or rows: 7 variables (columns) and 2389 rows
2. Variable list, including full names and definitions: 
- IdPlot: unique ID for each circular vegetation quadrat per year (area = 4 m2)   	
- IdSite: unique ID for each site (area = ~3 km2)  	
- IdPair: unique ID for each pair of exclosure/control plots (area of each plot = 80m2; area of each pair = 160 m2)  	
- Exclosure: fencing treatment (NO = absence of exclosure or control; YES = presence of exclosure)	
- Quadrat: number of each circular vegetation quadrat within each exclosure or control plot.	
- IdQuadrat: unique ID for each circular vegetation quadrat, irrespective of the year (area = 4 m2) 	
- Year: year of sampling
